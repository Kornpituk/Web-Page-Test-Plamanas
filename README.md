## üß± HTML Structure

### Header
- Navbar
  - Plaimanas logo
  - Navigation links
  - EDITORIAL dropdown
  - Currency selector
  - Hamburger menu (mobile)

### Main Sections

1. **Hero**
   - Full-width image/video
   - Brand name overlay

2. **NEW Section**
   - Image + descriptive text

3. **BESTSELLER**
   - Full-width banner or video background

4. **COLLECTION Grid**
   - 1 large feature item
   - 2 smaller items
   - Hover image/video swap

5. **Marquee Ticker**
   - Infinite scrolling text  
   - `NEW ARRIVAL ‚Üê BEST SELLING ‚Üí`

6. **Features / Stories**
   - Worldwide Shipping
   - Customer Service
   - Secure Payment

7. **FAQ**
   - Category tabs
   - Accordion interaction

8. **Inquiry Form**
   - Onsite / Hybrid radio selection
   - Input fields
   - Validation

### Footer
- Multi-column links
- Newsletter form
- Homepage link
- Brand information
